1. Enter songs you want to find, each song on a separate line
2. Be descriptive - artist name + song on each line
3. Press button

Files are downloaded in the default location of chrome's downloads: C:\Users\<USER>\Downloads.
Requires chromedriver, for now only works with chrome.
